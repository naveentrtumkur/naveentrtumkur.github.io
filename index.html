Graduate Research Assistant

May 2018 - Present

The Ohio State University

- Our current research involves Autonomously capturing good quality Selfie using A*, KNN, energy A* by using Utility assigned while an image is captured.
- A utility is a normalized value between 0 and 1 which is calculated based on 54 features extracted from Drone's current position and the image taken.
- We have benchmarked the drawbacks of using OpenCV's LBP model and the strengths of using DLIb (State-of-the-art CNN) for facial recognition. Our training data is Autonomy cube (offline 2x3x2 cubes captured) which will be used to predict efficient next position drone should move to get better Selfie.
- The beauty of our work is that we have developed a model correctly, so given Autonomy settings like Processors in edge device, battery of Drone, algorithm to be used, we can predict the motion and energy consumption without actually flying a Drone. This saves a lot of time for a User/Engineer.
